# 관리자 사용 가이드

이 문서는 비개발자 운영자 기준으로 `/studio`에서 콘텐츠를 수정하는 방법을 설명합니다.

## 1. 로그인

1. 배포된 사이트 주소 뒤에 `/studio`를 붙여 접속합니다.
2. `Sign in with Sanity`로 로그인합니다.
3. 권한이 부여된 계정이라면 바로 콘텐츠 편집 화면으로 이동합니다.

예시:

- `https://your-domain.vercel.app/studio`

## 2. 처음 한 번 해야 하는 작업

Studio에서 아래 문서를 생성해 주세요.

- `Site Settings` 1개
- `Portfolio Category` 여러 개
- `Portfolio` 여러 개
- `Video` 여러 개
- `Package / Price` 여러 개

`Site Settings`는 사이트 전체 공통 정보라서 보통 1개만 운영합니다.

## 3. 어떤 정보를 어디서 수정하나요?

### Site Settings

- 브랜드명
- 메인 배너 문구
- 메인 대표 이미지
- 메인 소개글
- About 소개글
- 연락처
- 이메일
- 주소
- 인스타그램 링크
- 카카오톡 상담 링크

### Portfolio Category

- 포트폴리오 분류명
- 슬러그
- 설명

### Portfolio

- 제목
- 슬러그
- 목록용 짧은 설명
- 상세 설명
- 촬영일
- 촬영 장소
- 대표 이미지
- 갤러리 이미지 여러 장
- 연결할 카테고리
- 메인 노출 여부

### Video

- 영상 제목
- 영상 설명
- 유튜브 URL
- 영상 대표 이미지
- 메인 노출 여부

### Package / Price

- 패키지명
- 가격
- 설명
- 포함 항목
- 추가 안내 문구
- 뱃지 문구
- 정렬 순서

## 4. 수정 후 반영

- Studio에서 저장하면 웹사이트에 자동 반영됩니다.
- 배포 환경에서는 보통 1분 이내에 최신 데이터가 갱신됩니다.
- 유튜브 링크나 가격만 바꿔도 별도 코드 수정은 필요 없습니다.

## 5. 운영 팁

- 대표 이미지는 가로 1600px 이상 권장
- 세로형 사진과 가로형 사진을 섞어도 사이트 레이아웃이 반응형으로 대응
- 포트폴리오 제목은 너무 길지 않게 작성
- 유튜브는 `youtube.com/watch?v=...` 또는 `youtu.be/...` 형식 모두 가능
- 카카오톡 채널 링크는 실제 상담용 링크로 입력

## 6. 주의 사항

- `Site Settings`는 여러 개 만들지 않는 것을 권장합니다.
- 실수로 잘못 저장한 경우, Sanity의 문서 히스토리에서 복구할 수 있습니다.
- 공개되지 않아야 하는 데이터는 입력하지 않는 것이 좋습니다.
